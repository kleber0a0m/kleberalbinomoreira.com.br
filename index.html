Software de gerenciamento feito em C# e MySQL

Como um projeto de extensão voluntário no IFSULDEMINAS eu e mais 2 colegas(Tiago Carvalho e Marcos Daniel Cardoso) desenvolvemos esta aplicação para a Cervart.
O objetivo desse software é gerir o trabalho e o controle tanto de acesso quanto de estoque de ingredientes e armazenamento de dados como receitas e fórmulas das bebidas que são produzidas na cervejaria.

Inicialmente foi realizado uma análise de requisitos, para identificar as necessidades do setor.
Para o desenvolvimento do software foi utilizada a linguagem de programação C# incluindo o módulo de Windows Forms e .NET. Para armazenar os dados que foi usado o banco de dados relacional MySQL.
Como metodologia no desenvolvimento do software foi usado como referência o manifesto ágil que valoriza algumas características como, indivíduos e interações mais do que processos e ferramentas e software em funcionamento mais do que documentação abrangente. 

Antes da implementação do software na Cervat (Cervejaria do IFSULDEMINAS Campus Machado) não era usado nenhum software de gestão para catalogar seus ingredientes e receitas ou gerir seu estoque e usuários, isso ocasionava vários problemas como atraso na produtividade, incoerência nos dados e perda de matéria prima. Sabendo dos benefícios dos softwares de gestão, foi identificada a necessidade de melhorar o controle geral na cervejaria e visto isso foi decidido criar um software de gerenciamento que visa melhorar o ambiente de trabalho, melhorar a eficiência da cervejaria reduzindo gastos e perdas em relação aos ingredientes que lá são utilizados.

As principais funções desse software são:

  • Controlar estoque de ingredientes e produtos.
  • Registrar as receitas de bebidas.
  • Gerenciar e supervisionar usuários e suas ações.

Aplicativo em C# de uma lista de compras

Aplicativo feito em C# usando Windows Forms, com banco de dados MySQL.

  • Cada usuário tem sua propria lista de compras.
